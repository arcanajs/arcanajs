# ArcanaJS Documentation

**ArcanaJS** is a modern, hybrid framework for building React applications with Node.js. It seamlessly combines the SEO benefits and initial load performance of **Server-Side Rendering (SSR)** with the fluid interactivity of a **Single Page Application (SPA)**.
